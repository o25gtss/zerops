Setup Guides

Android

  1. Use PowerTunnel or download Custom APK
  2. Click copy on PowerTunnel ‚Üí paste in app
  3. For Virtual Host: copy hosts entries
  4. Login to Growtopia!

iOS

  1. Install Surge 5 from App Store
  2. Copy Surge host ‚Üí Import Profile from URL
  3. Setup VPN ‚Üí Allow configuration
  4. Done, open Growtopia

Windows

  1. Press Win + R, type: C:\Windows\System32\drivers\etc
  2. Edit hosts file with Notepad
  3. Copy Windows hosts entries ‚Üí paste at bottom
  4. Save and login

Virtual Host

  1. Copy Virtual Host entries
  2. Open Virtual Host app on Android/PC
  3. Paste and apply
  4. Connect to server
